# MERN Stack Professional Portfolio 🚀

A high-performance, responsive portfolio website built to showcase my technical journey and projects.

## 🛠️ Tech Stack
- **Frontend:** React.js (Vite)
- **Backend:** Node.js & Express
- **Database:** MongoDB
- **Styling:** Modern CSS3 with Glassmorphism & Radial Gradients

## 🌟 Key Features
- **Seamless UI:** Edge-to-edge design with smooth background blending.
- **Dynamic Content:** Projects are fetched directly from a MongoDB database.
- **Mobile Responsive:** Optimized for all screen sizes.
- **Job Ready:** Direct links to LinkedIn, GitHub, and Email.
